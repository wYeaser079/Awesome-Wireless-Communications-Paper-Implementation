  Key design decisions

  1. CVX instead of YALMIP+MOSEK — more common in the wireless research
  community; still uses the same SCA-SOCP formulation line-for-line. MOSEK via
  CVX Professional is strongly recommended for speed.
  2. One canonical file per mathematical object (γ, SE, EE, SCA, selection).
  3. Small defaults + clear scale-up knobs — each figure script is set up to run
   in minutes on a laptop; comments indicate which loop lists to expand to reach
   the paper's published sizes.
  4. Verified formula detail: the paper's γ_{k,k'} uses β_{m,k}/β_{m,k′} (no
  square root) — I confirmed this from Appendix A's UI_{k,k'} derivation and the
   authors' own code, and coded it correctly.
