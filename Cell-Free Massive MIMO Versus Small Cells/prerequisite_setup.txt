● Checking Prerequisites

  1. MATLAB R2018a or later

  Run in MATLAB's command window:

  version    % shows your MATLAB version
  ver        % lists all installed toolboxes

  To specifically check for the Optimization Toolbox (needed for linprog):

  license('test', 'Optimization_Toolbox')   % returns 1 if licensed
  which linprog                              % shows path if available

  If linprog is missing, install the Optimization Toolbox via MATLAB's Add-On 
  Explorer (Home tab > Add-Ons > Get Add-Ons) or through your MathWorks account.

  2. CVX

  Check if CVX is installed:

  which cvx_begin   % shows path if CVX is on the MATLAB path
  cvx_version       % prints CVX version info

  To install CVX:

  1. Download from http://cvxr.com/cvx/download/
  2. Extract to a permanent folder (e.g., C:\cvx)
  3. In MATLAB, run:
  cd C:\cvx
  cvx_setup        % adds CVX to the path and verifies solvers
  savepath          % saves the path so it persists across sessions

  Since CVX is marked optional in the code, the implementation will fall back to
   a heuristic method for downlink power control if CVX isn't present — so it's
  not strictly required to run the simulation.

● The error shows MATLAB is treating "C:\cvx" as a relative path from your
  current directory. Two fixes:

  Option 1: Remove the quotes

  cd C:\cvx
  MATLAB's cd doesn't need quotes unless the path has spaces.

  Option 2: Use the function syntax

  cd('C:\cvx')

  If the path has spaces

  cd('C:\Program Files\cvx')

  Use the function-call form cd('...') with parentheses and single quotes —
  that's the reliable way in MATLAB. The cd "..." syntax with double quotes
  doesn't work the same way.
