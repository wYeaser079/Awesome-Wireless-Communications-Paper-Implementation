● Running Order

  The scripts are independent -- each one generates its own network, channels,
  and results from scratch. You can run them in any order. However, I recommend
  this sequence based on complexity (fastest to slowest):

  Step 1: main_figure4.m (run first)

  - Runtime: ~30-60 min (10 setups, 1000 realizations)

  - What it does: Generates Figure 5 (UL SE CDF comparison)
