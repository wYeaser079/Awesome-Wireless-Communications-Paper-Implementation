  ---
  Implementation Complete: 18 MATLAB Files

  File Descriptions

  Utility Functions (6 files)

  File: functionRlocalscattering.m
  Purpose: Generates spatial correlation matrices using the Gaussian local
    scattering model from [3, Sec. 2.6]. Takes ULA parameters (N antennas, angle

    of arrival, angular spread) and produces the N×N Hermitian correlation
  matrix
     via numerical integration.
  ────────────────────────────────────────
  File: generateSetup.m
  Purpose: Creates the complete network topology: 1×1 km area with L APs on a
    grid, 4 BSs on a 2×2 grid, K randomly placed UEs. Computes 3GPP Urban
    Microcell path loss (Eq. 37: -30.5 - 36.7*log10(d) with 4 dB shadow fading),

    spatial correlation matrices for both APs and BSs, and performs greedy pilot

    assignment minimizing contamination.
  ────────────────────────────────────────
  File: functionChannelEstimates.m
  Purpose: Generates correlated Rayleigh fading channel realizations via
    sqrtm(R)*CN(0,I) and computes MMSE channel estimates using Eqs. (4)-(6).
    Handles pilot contamination from co-pilot UEs sharing the same pilot index.
    Returns true channels, estimates, and error covariance matrices.
  ────────────────────────────────────────
  File: generateSetup_threeslope.m
  Purpose: Alternative setup using the COST-Hata three-slope model from Ngo et
    al. [16] for Figure 4 comparison. Different path loss, shadow fading (8 dB),

    and decorrelation distance (100 m).
  ────────────────────────────────────────
  File: pathloss_threeslope.m
  Purpose: Three-slope path loss model: constant for d<10m, 20 dB/decade for
    10-50m, 35 dB/decade for d≥50m (Eq. 39).
  ────────────────────────────────────────
  File: functionComputeSE_L2_LMMSE_ngo.m
  Purpose: Monte Carlo SE computation for Level 2 with L-MMSE combining under
  the
    three-slope model. Used in Figure 4 to show L-MMSE improvement over MR.

  Core SE Computation (4 files)

  File: functionComputeSE_AP_uplink.m
  Purpose: Central function. Computes uplink SE for all 4 cooperation levels
  with
    both MR and MMSE/L-MMSE combining, plus MMSE-SIC sum SE. Level 4 uses global

    MMSE combining (Eq. 13-14). Levels 3/2 use local L-MMSE with LSFD/uniform
    weights (Eqs. 16, 19-22). Level 1 selects max-SE AP (Eq. 26-27). Also
    computes MMSE-SIC via log2 det(I + P*Hhat'*E^{-1}*Hhat) (Proposition 4).
  ────────────────────────────────────────
  File: functionComputeSE_BS_uplink.m
  Purpose: Computes SE for cellular Massive MIMO with MR and M-MMSE combining
    (Eqs. 35-36). Each BS estimates channels from ALL UEs (including other
  cells)
     for M-MMSE. Assigns UEs to cells by strongest average channel.
  ────────────────────────────────────────
  File: functionComputeSE_CF_uplink_ngo.m
  Purpose: Closed-form SE for Level 2 with MR combining (N=1) matching Ngo et
    al.'s formulation.
  ────────────────────────────────────────
  File: functionComputeSE_SC_uplink_ngo.m
  Purpose: Closed-form SE for small cells using Proposition 3 (Eq. 29) with
    exponential integral E₁. Two AP selection strategies: max-β (Ngo's) and
    max-SE (improved).

  Power Optimization (2 files)

  File: functionSINRterms_uplink_ngo.m
  Purpose: Extracts SINR numerator/denominator terms (without power factors) for

    both cell-free and small-cell configurations. Used by the power optimizer.
  ────────────────────────────────────────
  File: functionPowerOptimization_maxmin.m
  Purpose: Max-min fairness power allocation via bisection + fixed-point
    iteration. Does NOT require CVX toolbox (unlike the original code).

  How to Run

  1. Open MATLAB and navigate to: MMSE Processing/codes/
  2. Run main.m and select which figure to generate (1-6), or:
    - Run individual scripts directly: simulationFigure2a_3, simulationFigure6,
  etc.
  3. Start with Figure 6 (instant, no simulation) to verify setup
  4. For faster results: reduce nbrOfSetups (e.g., 10) and nbrOfRealizations
  (e.g., 50) in each script
  5. For paper-quality results: set nbrOfSetups = 200 and nbrOfRealizations = 
  1000

  Execution Order (Recommended)

  Figure 6  → instant (analytical)
  Figure 4  → ~10 min (closed-form + small Monte Carlo)
  Figure 2a/3 → ~30 min (Monte Carlo, L=400, N=1)
  Figure 5  → ~30 min (similar to 2a/3 + SIC)
  Figure 2b → ~1-2 hours (N=4 increases computation significantly)

  Dependencies

  - MATLAB R2018b or later (uses expint, chol, standard functions)
  - No external toolboxes required (power optimization uses bisection instead of
   CVX)

  Key Design Decisions

  - Noise variance is normalized to 1 (channel gains include gain-over-noise)
  - Wraparound topology eliminates edge effects
  - Pilot assignment uses greedy contamination minimization
  - Default parameters are reduced for manageable computation time
